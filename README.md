# dat2csv

## Build Requirements

- [Qlot](https://github.com/fukamachi/qlot)
- make
- SBCL

## Usage
```
NAME:
  dat2csv - converts Concordance DAT file to CSV

USAGE:
  dat2csv [OPTIONS] <DAT-FILE>

OPTIONS:
      --help     display usage information and exit
      --version  display version and exit
  -h             help

AUTHORS:
  Jin-Ho King <j@kingesq.us>
```
